
# How to use Markdown to make reports

### Cách làm 

- hình ảnh minh hoạ gom vào 1 thư mục, ví dụ `screenshot` 
   - hình ảnh này sẽ được lưu trữ github (hoặc gitlab)
   - đính kèm vào file markdown bằng đường dẫn tương đối, ví dụ [mdwithVSCode.png](/screenshot/mdwithVSCode.png)
   - thay thế bằng hình mới khi có thay đổi  
- hình minh hoạ kèm theo từng bước thực hiện
   - hiện hình ảnh ngay trong file .md, ví dụ 
   ![mdwithVSCode.png](/screenshot/mdwithVSCode.png)
   - rút gọn hình dưới dạng link, nguời xem mở ra khi cần thiết
